## 🎯 Learning Approach

For every topic, I follow the same learning process:

1. Learn the theory.
2. Understand the algorithm.
3. Write pseudocode.
4. Implement the solution in Python.
5. Analyze time and space complexity.
6. Practice interview-style questions.
7. Document my learning.

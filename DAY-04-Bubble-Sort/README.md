# Bubble Sort

This folder contains my implementation of the **Bubble Sort** algorithm as part of my Data Structures and Algorithms (DSA) interview preparation.

## Files

* `easy.py` – Basic Bubble Sort implementation.
* `medium.py` – Optimized Bubble Sort using the `swapped` flag.
* `notes.md` – Theory, time complexity, and interview notes.

## Algorithm

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest unsorted element moves to the end of the array.

## Time Complexity

| Case    | Complexity |
| ------- | ---------- |
| Best    | O(n)       |
| Average | O(n²)      |
| Worst   | O(n²)      |

## Space Complexity

* O(1)

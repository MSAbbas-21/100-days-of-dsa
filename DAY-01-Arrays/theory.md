# Arrays

## What is an Array?

An array is a collection of elements stored in a sequence.

In Python, lists are used as dynamic arrays.

Example

arr = [10, 20, 30]

---

## Why do we use Arrays?

- Store multiple values in one variable
- Easy to access elements using an index
- Easy to traverse using loops

---

## Array Traversal

Traversal means visiting every element one by one.

Example:

for num in arr:
    print(num)

or

for i in range(len(arr)):
    print(arr[i])

---

## Big O Notation

Big O tells how efficient an algorithm is.

O(1)
- Accessing an element by index

O(n)
- Traversing an array

O(n²)
- Nested loops

---

## Time Complexity

Largest Element
- Time: O(n)
- Space: O(1)

Second Largest
- Time: O(n)
- Space: O(1)
